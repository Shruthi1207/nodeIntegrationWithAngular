Tool provides following capabilities


The list below identifies capabilities for initial release of this tool. This tool provides…

  1. ...directional recommendation of whether or not a Best Buy application can be hosted on a Public Cloud. This recommendation will be based on cloud adoption barriers and the application team's migration goals.
  2. ...a rough monthly cost of hosting the application in a Best Buy data center compared to a Cloud Service Provider (CSP) in a like-for-like comparison. The comparison will be based on categories and units defined by Best Buy (please see the Assumptions section for categories and units).
  3. …an estimate of potential savings when utilizing auto-scaling features available from major CSPs to accommodate spikes in workload demand (eliminate overprovisioning through "Replatforming for Elasticity").
  4. …an estimate of potential savings when selecting DBaaS (PaaS) for database hosting as an alternative to hosting DB on virtual machine instances (IaaS).
  5. …an estimate of cost of depreciation for both compute and storage based on a Best Buy provided start date and a 5 year refresh cycle.
  6. …an estimate of potential savings when selecting reserved vs on-demand instances for the compute part of hosting on the cloud.

Tool does not address following capabilities


  1. Application modernization and associated future cost savings are not included in cost calculations.
  2. This tool does not include any cost of transformation of the application or work required to perform the migration itself.
  3. Any recommendations and calculations not explicitly included or described under Tool Capabilities section are not addressed by the tool.

Assumptions


Cloud Barriers

  1. The cloud barriers defined in the tool are fixed.
  2. Security, Legal, Data, and any other relevant governance teams have previously approved the usage of the cloud as a new hosting platform and provided approval for the application to be hosted in a specific CSP.
  3. Any additional services such as Infrastructure, Middleware, Monitoring, Security, and IAM already exist in the cloud.

Cloud Benefits

  1. The Cloud benefits defined in the tool are fixed.

Hosting Calculator


Monthly Hosting Cost is based on following 4 categories and their corresponding units:
  1. Compute
    - calculations are based on predefined list of VM types based on memory size and number of cores.
    - Elastic infrastructure costs will be calculated for the year and averaged per month.
  2. Storage
    - calculations are based on the type and amount of consumed storage (in GB) by the application.
    - The storage pricing will be rounded up to the closest available pre-defined size of the SSD drives available on each CSP.
    - Performance requirements (IOPS) are not being considered and a performance assessment is not being conducted; only the capacity requirements are used for storage cost calculations.
    - All Best Buy SAN storage (SSD and non-SSD) will be matched to CSP's SSD Tier Pricing as requested by Best Buy.
  3. Network
    - calculations are based on a per VM cost defined by Best Buy as one Network Unit. Since CSPs determine network cost based on the size of outgoing data/traffic, Best Buy will provide an estimate for amount of outgoing traffic per application per month rounded to predefined ranges.
    - Cloud network services are calculated based on estimates provided by Best Buy.
    - The cost of shared network services in the cloud (Direct Connect, Gateways, etc.) is not included. The final network cost per VM maybe higher or lower due to distributed shared cost.
  4. Database
    - calculations are based on licensing model for each DB technology. SQL and Oracle: cost per core. Cassandra and Hadoop: cost per host. Since Best Buy will utilize a Bring Your Own License option, pricing for DB is only collected and used to provide basis for savings achieved when migrating to DBaaS.
  5. This tool will account for "On Demand" and "Reserved Instances" virtual machines (average discount percentages for each cloud provider). Other cloud provider discounts may be applicable, but will not be included in this calculation. Best Buy will have an option to adjust the average discount percentage.
  6. Operational support cost will remain the same as current cost at Best Buy, regardless of the deployment model.
  7. Application software licensing cost (except for the OS) will be considered the same as current cost at Best Buy, regardless of the cloud provider chosen.
  8. Network in the cloud is software-defined and not all traditional tools may continue working so redesign/adjustment may be required. This tool does not account for the cost of the redesign/adjustment.
  9. Monthly infrastructure cost will vary after performing application modernization.

Summary


  1. Total cost estimate is based on data provided by Best Buy.
  2. Application decision matrix will not differentiate between "Best Buy data center" and "Private Cloud". While virtualized, the Best Buy data center will not be considered as a Private Cloud. This tool is not considering Private Cloud as a deployment model for any of the cloud services.
